# ХИП КЛИНИК

Уебсайт на ХИП КЛИНИК — специализиран център за тазобедрената става в София.

Статичен сайт (HTML/CSS/JS, без зависимости).

- `index.html` — целият сайт (текст, стилове и скриптове в един файл)
- `hero-video.mp4` — видео фон (операционна), компресирано за уеб (~10 MB)

Секцията „За клиниката" вгражда Google Maps (УСБАЛО „Проф. Бойчо Бойчев"). Услугите и екипът са текстови — без външни изображения.

## Локално стартиране

```bash
python3 -m http.server 8000
# отвори http://localhost:8000
```
